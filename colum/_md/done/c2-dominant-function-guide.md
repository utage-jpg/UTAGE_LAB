# あなたの「主機能」はどれ？ 簡単な見分け方

MBTIを学び始めると、すぐに「認知機能」という言葉に出会います。そして、その中で「主機能」は、あなたの思考・行動の中心にある機能です。

でも正直、最初はよくわからないですよね。「FeとTiって何が違うの？」という状態から、自分の主機能を見つけるための手がかりを、この記事でお伝えします。

---

## 主機能とは、あなたの「一番の得意技」

人には8つの認知機能があります。ただし、8つ全員に均等に働いているわけではありません。一番よく使う、一番自然に発揮される機能——それが「主機能」です。

主機能は次の8種類のどれかです。

- **Fe**（外向的感情）：場の空気を読んで、みんなが心地よい状態にしようとする
- **Fi**（内向的感情）：自分の価値観と正直に向き合い、自分にとって正しいことをやり続ける
- **Te**（外向的思考）：効率と結果を重視し、論理的に物事を組織・推進する
- **Ti**（内向的思考）：物事の構造や仕組みを徹底的に理解しようとする
- **Ne**（外向的直観）：可能性やアイデアを広げ、点と点をつなぐことが得意
- **Ni**（内向的直観）：未来を見通し、本質をパターンとして直観的に把握する
- **Se**（外向的感覚）：今この瞬間の刺激・体験・行動に没頭する
- **Si**（内向的感覚）：過去の経験・記憶の積み重ねを大切にし、安定を好む

## 自分の主機能を見つける3つの質問

難しく考えなくて大丈夫です。以下の質問に正直に答えてみてください。

### 質問1：「何をしているときに一番生き生きするか」

- 人の感情に寄り添い、場を和ませるとき → **Fe**
- 自分の信念のために誰かに本当のことを言えるとき → **Fi**
- 計画を立てて、チームを動かして成果を出すとき → **Te**
- 複雑な問題の「本当の仕組み」が理解できたとき → **Ti**
- 新しいアイデアがどんどん出てきて止まらないとき → **Ne**
- まだ起きていない未来がなんとなく「見える」感じがするとき → **Ni**
- 何かを行動し、今この瞬間に没頭しているとき → **Se**
- いつものルーティンを丁寧にこなせているとき → **Si**

### 質問2：「何を失うと一番つらいか」

主機能は、使えない状況で強いストレスを感じます。

- 場の調和が崩れ、誰かが傷ついているのに何もできないとき → **Fe**
- 自分の価値観を裏切ること、強制されるとき → **Fi**
- 非効率なやり方を変えられず、成果が出ないとき → **Te**
- なんとなくで物事が決まり、根拠が何もないとき → **Ti**
- 可能性をすべて潰されて、同じことを繰り返すだけのとき → **Ne**
- 深く考える時間を奪われ、表面的なことしかできないとき → **Ni**
- 行動できるのに、ただ待つだけの状況が続くとき → **Se**
- 突然の変化を押しつけられ、準備する時間がないとき → **Si**

### 質問3：「他人からどんな人だと言われることが多いか」

- 「気難しいでしょ、でもなんか場の空気をよく読む」 → **Fe**
- 「良さそうだけど、感情的になりにくい？」 → **Fi**
- 「仕事できそう、判断が速い」 → **Te**
- 「屁理屈っぽい、何でも分析する」 → **Ti**
- 「発想が面白い、話が飛ぶ」 → **Ne**
- 「落ち着いている、何か考えていそう」 → **Ni**
- 「行動力がある、飽き性」 → **Se**
- 「真面目、几帳面、几帳面すぎる」 → **Si**

## タイプと主機能の対応表

MBTIのタイプが決まれば、主機能は自動的に決まります。

| タイプ | 主機能 |
|--------|--------|
| ENFJ / ESFJ | Fe（外向的感情） |
| INFP / ISFP | Fi（内向的感情） |
| ENTJ / ESTJ | Te（外向的思考） |
| INTP / ISTP | Ti（内向的思考） |
| ENTP / ENFP | Ne（外向的直観） |
| INTJ / INFJ | Ni（内向的直観） |
| ESTP / ESFP | Se（外向的感覚） |
| ISTJ / ISFJ | Si（内向的感覚） |

## 「主機能がわからない」人へ

自分の主機能がどれかわからない場合、以下の可能性があります。

**1. ストレスや抑圧で主機能が出づらくなっている**
長年「こうすべき」という外圧の中で生きてきたとき、本来の主機能が隠れることがあります。「本当に自分がしたいこと」を思い出してみてください。

**2. テスト結果と実際のタイプがずれている**
MBTIのオンラインテストは、あくまで目安です。特に「E/I」「J/P」のスコアが中間値に近い場合、テスト結果が不安定になりやすい。認知機能から逆引きする方が精度が上がることもあります。

**3. まだ主機能が育っていない**
若い頃は主機能がまだ洗練されていないことも。年齢を重ねるほど、自分らしく際立ってくる人が多いです。

## まとめ

主機能を知ることは、「自分が何に喜びを感じ、何に傷つきやすいか」を理解することです。これは自己分析の入口であり、同時に他者を理解する入口でもあります。

まず「一番生き生きすること」という問いを浮かべてみてください。そこに、あなたの主機能があります。
