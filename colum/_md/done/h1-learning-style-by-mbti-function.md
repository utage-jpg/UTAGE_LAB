# 暗記が得意なタイプ、構造理解が得意なタイプ

「自分は頭が悪い」「暗記力がない」——そう感じている人は少なくありません。でも、それは本当に能力の問題なのでしょうか。MBTIの認知機能では、情報をどのように処理するか、何を「覚えやすい形」と感じるかが、機能スタックによって大きく異なります。

## 学習の得意・不得意は努力だけの問題ではない

得意な学習法と苦手な学習法は、努力の量ではなく、認知機能の回路の違いから来ていることが多いのです。「自分はなぜこの勉強法だとうまくいかないのか」という問いへの答えが、機能スタックの中にあることが少なくありません。

## 各機能の学習スタイル

### Siタイプ——繰り返しと反復で定着
Si（内向的感覚）が強いタイプは、繰り返しと反復による定着を得意とします。毎日同じ時間に同じ手順で勉強する、単語帳を何周もするといった方法が最も効果的です。急な変更や体系のない雑多な情報には対応しにくいことがあります。

### Niタイプ——本質・構造を先につかむ
Ni（内向的直観）が強いタイプは、全体の構造や本質的なパターンをつかもうとします。目次を眺めて全体像を把握してから細部に入る方が情報が定着しやすいです。構造が見えないまま細部を暗記させられると著しく効率が落ちます。

### Tiタイプ——論理の構造として整理
Ti（内向的思考）が強いタイプは、「なぜそうなるのか」という論理的な因果関係の網に組み込むことで理解します。「このルールはこの原理から導ける」という構造が見えると、暗記しなくても再導出できます。

### Neタイプ——アナロジーと連想で覚える
Ne（外向的直観）が強いタイプは、アナロジーや連想によって情報をつなげます。知識の横断や応用が許される学習環境の方が能力を発揮します。

### Seタイプ——体験と実践で学ぶ
Se（外向的感覚）が強いタイプは、実際にやってみることで最も早く学習が進みます。体験型・実習型の学習では驚くほど吸収が速いです。

## 機能別・学習スタイル比較表

| 主な機能 | 得意な学習法 | 苦手なパターン |
|---|---|---|
| Si（内向的感覚） | 反復・繰り返し・手順の確認 | 体系のない雑多な情報。急な変更 |
| Ni（内向的直観） | 全体構造・本質の把握から細部へ | 理由なき丸暗記 |
| Ti（内向的思考） | 因果関係・論理構造の理解 | 「なぜそうなるか」が説明されない暗記 |
| Ne（外向的直観） | アナロジー・比喩・連想。横断的な学び | 単線的な反復暗記 |
| Se（外向的感覚） | 実践・体験・ロールプレイ | 長時間の座学・抽象的な理論 |
| Te（外向的思考） | 目標・成果基準が明確な学習 | 曖昧なゴール・意義が見えない作業 |

## 学習スタイル・認知機能マトリクス

- 暗記重視 × インプット型：Si（反復・手順定着型）
- 概念重視 × インプット型：Ni（全体構造把握型）
- 暗記重視 × アウトプット型：Se（実践・体験型）
- 概念重視 × アウトプット型：Ti / Ne（論理・連想型）

## まとめ

「暗記が苦手」は頭の悪さではなく、あなたの認知機能が別の回路を好んでいるサインかもしれません。自分の認知機能を知ることで、自分に合った学習法を選べるようになります。
